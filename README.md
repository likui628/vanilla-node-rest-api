# vanilla-node-rest-api


## api

[postman collection file](./vanilla-node-rest-api.postman_collection.json)

```
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id
```

## usage
```
# install
pnpm install

# run develpment
pnpm run dev

# run production
pnpm run start
```
